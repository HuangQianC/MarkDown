# 其他
- 斜体：一对星号，比如：*斜体*
- 加粗：两对星号，比如：**加粗**
- 编写行内代码：一对反引号，比如，`printf()`
- 下划线：使用html中的u标签，比如，<u>下划线</u>
- 表情：一个冒号 + 名称，比如，😠
- 下标：使用html中的sub标签，比如，H<sub>2</sub>O
- 上标：使用html中的sup标签，比如，X<sup>2</sup>
- 高亮：使用html中的mark标签，比如：<mark>高亮文本</mark>
